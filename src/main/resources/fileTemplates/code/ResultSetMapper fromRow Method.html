This template is provided by the ResultSetMapper plugin. It is used each time a fromRow action is executed.
List of custom variables
${packageName}   Target package name
${generatorName}   Name of the code generator
${j2eNamespace}   Project J2EE namespace: javax or jakarta
${mapper}   FromRowMapper or ResultSetMapper
${class}   PsiClass target
${className}   Simple name of the class target
${fields}   List of available class fields
List of field variables
${isValid}   true if field can be mapped
${isArray}   true if field is an array type
${isList}   true if field is a java.util.List
${isSet}   true if field is a java.util.Set
${isCollection}   true if field is a java.util.Collection
${isInitialized}   true if field has an initializer
${isPrimitive}   true if field has a primitive type
${field}   PsiField
${type}   PsiField.type
${name}   PsiField.name
${componentType}   PsiType of array component
${collectionType}   PsiType of list, set or collection type parameter
${columnType}   Mapped SQL column type name
${columnName}   Expected SQL column name
